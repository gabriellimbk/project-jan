How to use with Canvas

  1. In Canvas, go to Settings > Apps.
  2. Search for and add the Redirect Tool.
  3. Enter the URL of this page.
  4. In the "Custom Fields" or LTI setup, use substitutions like: ?id=$Canvas.user.id or ?user_id=$Canvas.user.id
  5. Update the data-target-url in this file to your app URL.
